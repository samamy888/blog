# wrap()替換dom節點

最近要處理某個dom把他們都套入div時發現有點棘手，與朋友討論後發現有jquery有wrap()相關語法，特地寫起來筆記一下😅

# wrap()

在每個匹配的元素外層包上html元素

假設我們目前有

<p>foo</p>
1

要給這段p元素套上div時 就可以用wrap()

$('p').wrap('<div class="new" />');
1

那最後的結果就會這樣

<div class="new">
    <p>foo</p>
</div>
1
2
3

如果想要自訂片段也可以用以下方式:

$('p').wrap(function() {
  return '<div class="' + $(this).text() + '" />';
});
1
2
3

結果:

<div class="foo">
    <p>foo</p>
</div>
1
2
3

# wrapInner()

在每個匹配的元素內層寫入html元素

假設我們目前有

<div>foo</div>
1

要給這段div元素加入b時 就可以用wrapInner()

$('div').wrapInner('<b/>');
1

那最後的結果就會這樣

<div>
    <b>foo</b>
</div>
1
2
3

也可以自訂:

$('div').wrapInner(function() {
  return '<span class="' + $(this).text() + '" />';
});
1
2
3

結果:

<div>
    <span class="foo">foo</span>
</div>
1
2
3

# wrapAll()

在所有匹配的元素外層包上html元素

假設我們目前有

<p>foo</p>
<p>bar</p>
1
2

要在這兩段外面用個div包起來時，可以這樣寫:

$('p').wrapAll('<div class="new" />');
1

那最後的結果就會這樣

<div class="new">
    <p>foo</p>
    <p>bar</p>
</div>
1
2
3
4

只是wrapAll就無法自訂片段了！！

# unwrap()

將匹配到的父集合刪除，保留自己跟兄弟元素在原來的位置

假設我們目前有

<div>
    <p>foo</p>
</div>
1
2
3

假設要刪除div並保留p段落的時候，可以這樣寫:

$('p').unwrap();
1

那最後的結果就會這樣

<p>foo</p>
1

以上就是wrap相關使用技術了~
